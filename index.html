Resumen de XML

¿Qué es XML?

XML (extensible Markup language) es un lenguaje creado para almacenar, organizae y transportar datos. No se usa para diseñar ni para crear páginas web visuales;
su objetivo es describir información usando etiquetas. Se parece a HTML, pero la gran diferencia es que tú puedes crear tus propias etiquetas.

¿Para qué se usa XML?

• Intercambio de datos entre sistemas. Cuando un sistema necesita enviar información de un sistema a otro,
XML es la manera estándar y clara de hacerlo.
• Archivos de configuración. Muchos programas guardan sus ajustes en XML, porque es fácil de leer y modificar.
• Almacenar datos estructurados. Sirve para representar información jerárquica, en forma de árbol.
• Servicios web. Antes de que JSON fuera popular, XML era el formato principal para APIs.
• Documentos y bases de datos. Algunas bases de datos permiten almacenar datos directamente en XML

Características importantes.

• Usa etiquetas personalizadas.
• Se lee bien por humanos y por computadoras.
• Mantiene los datos organizados.
• Es flexible y extensible.
• No define diseño, solo estructura.

ejemplo:

            <persona>
            <nombre>Karen</nombre>
            <edad>21</edad>
            <ocupacion>Estudiante</ocupacion>
            <contacto>
                <email>karen@example.com</email>
                <telefono>809-555-1234</telefono>
            </contacto
        </persona>
